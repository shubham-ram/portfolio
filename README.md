# Shubham Ram — Interactive Developer Portfolio

A creative, interactive portfolio showcasing my work as a **Full-Stack Engineer** and **UI Architect**. Built with modern web technologies, focused on fluid animations, glassmorphic UI, and pixel-perfect performance.

🌐 **Live:** [shubham-ram.com](http://shubham-ram.com/)

## 🌟 Features
- **Fluid Animations:** Powered by Framer Motion with smooth transitions and micro-interactions.
- **Glassmorphic UI:** Modern aesthetic with dynamic blurs and gradient meshes.
- **Interactive Sound Design:** Web Audio API integration for subtle UI sound effects.
- **Responsive Design:** Crafted to work seamlessly across all device sizes.
- **Performance Optimized:** Built on Vite + React 19 for fast load times.

## 🚦 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/shubham-ram/portfolio.git
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm dev
   ```
4. Build for production:
   ```bash
   pnpm build
   ```
