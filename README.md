# Shubham - Interactive Developer Portfolio

A creative, interactive, and futuristic portfolio showcasing my work as a **Frontend Engineer**, **UI Architect**, and **Creative Developer**. Built with modern web technologies, focusing on fluid animations, spatial UI concepts, and pixel-perfect performance.

## 🌟 Features
- **Fluid Animations:** Powered by Framer Motion, featuring smooth transitions and micro-interactions.
- **Glassmorphic UI:** Modern aesthetic with dynamic blurs and gradient meshes.
- **Interactive Sound Design:** Web Audio API integration for subtle UI sound effects.
- **Responsive Design:** Carefully crafted to work seamlessly across all device sizes.
- **Performance Optimized:** Built on Vite + React 19 for extremely fast load times.

## 🚦 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/shubham-ram/portfolio.git
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm dev
   ```
4. Build for production:
   ```bash
   pnpm build
   ```
